# IR40 Client for Huawei MatePad SE 11

Нативный Android-клиент для лазерной рулетки **FNIRSI-IR40**.

## Возможности

- поиск IR40 по BLE-имени или сервису `EE01`;
- подключение к GATT-сервису `0000ee01-0000-1000-8000-00805f9b34fb`;
- подписка на результаты через `EE02`;
- команды старта и остановки непрерывного измерения через `EE03`;
- разбор линейных, площадных и других пакетов с сохранением сырых HEX-данных;
- журнал до 200 измерений;
- копирование последнего результата в JSON;
- экспорт журнала в CSV через системный файловый диалог;
- никаких Google Play Services и сетевых разрешений.

## Совместимость

- Android 8.0+ (`minSdk 26`);
- Android 12+ permissions `BLUETOOTH_SCAN` / `BLUETOOTH_CONNECT`;
- адаптивная компоновка для экранов планшетов `sw600dp`.

На HarmonyOS NEXT / HarmonyOS 5+ без Android-совместимости APK не устанавливается — для такой системы потребуется отдельная сборка HAP/APP на ArkTS.

## Протокол

- Service: `0000ee01-0000-1000-8000-00805f9b34fb`
- Notify: `0000ee02-0000-1000-8000-00805f9b34fb`
- Write: `0000ee03-0000-1000-8000-00805f9b34fb`
- Start continuous: `00 07 02 08 0E 00 00 00 01`
- Stop continuous: `00 07 02 08 0E 00 00 00 00`

Числовой масштаб `1/1000` подтверждён опубликованными пакетами в метрическом режиме. Приложение всегда показывает исходный HEX-пакет, чтобы можно было проверить другие прошивки и единицы.

## Сборка

```bash
gradle testDebugUnitTest assembleDebug
```

APK появится в `app/build/outputs/apk/debug/app-debug.apk`.
